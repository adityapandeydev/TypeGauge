# TypeGauge

TypeGauge is a project inspired by the popular typing game [TypeRacer](https://play.typeracer.com/). This initial build focuses on learning and implementing state management in a React environment. The UI is currently a direct replication of the TypeRacer site, serving as a starting point for further enhancements.
